https://github.com/mjtorn/dotnvim
=================================

mjt's .condig/nvim/ stuff

See also `https://github.com/mjtorn/dotvim`

Installation
------------

    git clone git://github.com/mjtorn/dotnvim.git .config/vim/

    cd .config/nvim/

    git submodule update --init

    mkdir undo
