dotvim
======

mjt's .vim stuff

Installation
------------

    git clone git://github.com/mjtorn/dotvim.git .vim/

    ln -s .vim/vimrc .vimrc

    cd .vim/

    git submodule update --init

    mkdir undo

Bonuses
-------

Some flake8 conf for use with syntastic: https://gist.github.com/mjtorn/9870434
